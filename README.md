# framer-fluid-effect

A configurable WebGL fluid background built with [webgl-fluid-enhanced](https://github.com/michaelbrusegard/WebGL-Fluid-Enhanced) — the same engine used by Framer's Fluid Animation component.

Tune colors, splat strength, bloom, and more in the live configurator, then copy the snippet for React or Framer.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Usage

- **React / Next.js:** Click **Copy for React** and drop the component into your app. Run `npm install webgl-fluid-enhanced`.
- **Framer:** Click **Copy for Framer**, create a Code Component in Assets → Code, paste the snippet, and install `webgl-fluid-enhanced` when prompted.

## Stack

- Next.js (App Router) + React + TypeScript
- [webgl-fluid-enhanced](https://www.npmjs.com/package/webgl-fluid-enhanced)
